# 📧 Spam Email Classifier

A simple Spam Email Classifier developed using Python and Flask as part of the CodeTech Technologies Internship. This application classifies an email as **Spam** or **Not Spam** based on common spam keywords.

## 🚀 Features

- Detects Spam Emails
- Identifies Non-Spam Emails
- Simple and User-Friendly Interface
- Fast Email Classification

## 🛠️ Built With

- Python
- Flask
- HTML5
- CSS3

## 📁 Project Structure

```
spam_email_classifier/
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
├── README.md
└── Screenshot.png
```

## ▶️ How to Run

1. Install the required packages:

```
pip install -r requirements.txt
```

2. Run the application:

```
python app.py
```

3. Open your browser and visit:

```
http://127.0.0.1:5000
```
